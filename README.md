# Decimal-Binary Converter

This is a simple Python program that converts decimal numbers to binary and binary numbers to decimal.

## Usage

1. Run the program.
2. Choose an option:
    - `1` for Decimal to Binary conversion
    - `2` for Binary to Decimal conversion
3. Follow the prompts to input your number.

### Example

#### Decimal to Binary
```
Choose an option: 
 1. Decimal to binary 
 2. Binary to decimal
 Option: 1
Input your decimal number:
Decimal: 10
Binary: 1010
```

#### Binary to Decimal
```
Choose an option: 
 1. Decimal to binary 
 2. Binary to decimal
 Option: 2
Input your binary number:
Binary: 1010
Decimal: 10
```

## Requirements

- Python 3.x
